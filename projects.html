FinderOnline

FinderOnline is an innovative healthcare platform designed to connect users with specialist doctors for seamless medical consultations. The application provides a convenient way for users to browse through a comprehensive list of healthcare professionals, view their specialties, and make informed decisions when booking appointments.

  • Doctor Listings: Users can search for and view detailed profiles of specialist doctors, including their expertise, availability, and consultation fees.
  • Appointment Booking: The platform allows users to select their preferred date and time for consultations, ensuring a personalized and efficient scheduling experience.
  • Booking Requests: Once a user selects a time slot, a booking request is sent directly to the doctor for confirmation.
  • Integrated Communication: FinderOnline supports seamless communication through:
    • Text Chat: For quick messaging between users and doctors.
    • Audio Calls: Providing a more personal interaction for detailed discussions.
    • Video Calls: Leveraging the Twilio library for high-quality face-to-face consultations.

The application is developed using Firebase for real-time data synchronization and authentication, ensuring a secure and responsive user experience. Java powers the core functionalities, while Twilio handles robust audio and video communication.

Tech: Firebase, Twilio, Java.

Eatlub

Eatlub is a comprehensive food delivery solution designed for both restaurants and riders. The app allows restaurants to register, list their food items, and manage orders. Users can browse menus, select food, track their order history, rate riders, and tip them for good service. Riders can accept orders, track deliveries, and ensure timely service. The backend of the app uses REST APIs built in Java with the Spring MVC framework, providing a robust and scalable solution for both restaurants and riders.

Technologies Used:

  • Google Maps API: Used for tracking riders and optimizing delivery routes.
  • Firebase: Used for real-time notifications to users and riders.

Reading Glasses

Reading Glasses is an innovative app that helps people with low vision or reading difficulties. It allows users to magnify text and convert it into a more readable format. Users can upload images, and the app will extract the text using Optical Character Recognition (OCR) technology. The app also includes a text-to-speech feature powered by Google Text-to-Speech, allowing users to listen to the text. Additionally, users can save the converted text as a file for later use.

Technologies Used:

  • OCR (Tesseract SDK): Used for extracting text from images.
  • Google Text-to-Speech API: Converts text into speech.
